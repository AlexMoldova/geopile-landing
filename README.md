# Geopile — Landing Page pentru Facebook Ads

Landing page în stilul concria.kiev.ua — design curat, schemă verde, optimizat pentru conversii maxime din Facebook Ads.

## Despre

Site single-page pentru **Geopile** — companie specializată în fundații pe piloți prefabricați din beton prin presare statică (Constanța, România).

## Design

Inspirat din [concria.kiev.ua](https://concria.kiev.ua/ru):
- 🟢 Schema de culori verde (#116b35 / #4fb725 / #14c947)
- Layout curat, secțiuni distincte cu fundaluri alternante
- Carduri cu border hover effects
- Tabel comparativ cu tipuri de fundații
- Algoritm de lucru în 6 pași
- Galerie foto din proiecte reale Geopile

## Caracteristici

- ✅ **Mobile-first** — sticky CTA bar pe mobil
- ✅ **Limba română** — tot conținutul adaptat
- ✅ **Meta Pixel** integrat (PageView, Lead, Contact, Scroll50, Scroll90)
- ✅ **Thank you page** pentru tracking conversii
- ✅ **Formular lead** — nume, telefon, email, locație, tip construcție
- ✅ **Foto din geopile.eu** — proiecte reale 2021-2025
- ✅ **Header sticky** cu navigație și telefon
- ✅ **Tabel comparativ** — piloți presați vs bandă vs forati vs elicoidali
- ✅ **SEO** — Open Graph, meta tags, semantic HTML

## Secțiuni

1. **Header** — logo Geopile + nav + telefon
2. **Hero** — fundații pe piloți + CTA dual
3. **Trust bar** — 7+ ani, 250+ proiecte, 50 ani garanție
4. **6 avantaje** — ca la concria (speed, economic, durability etc.)
5. **Tipuri de piloți** — secțiuni, spec, foto
6. **Tabel comparativ** — piloți presați vs 3 alte tipuri
7. **Aplicații** — 6 domenii de utilizare
8. **Servicii** — 6 servicii complete
9. **Algoritm de lucru** — 6 pași
10. **Portofoliu** — 8 foto proiecte din geopile.eu
11. **Stats bar** — cifre + feature list
12. **Despre noi** — descriere Geopile
13. **Formular lead** — captură date
14. **Footer** — contact, servicii, social

## Setup

1. Înlocuiește `YOUR_PIXEL_ID_HERE` în `index.html` și `multumim.html`
2. Deploy pe GitHub Pages
3. Lansează campania Facebook cu link către site

## Deploy

```bash
git add . && git commit -m "Redesign: concria style" && git push
```
